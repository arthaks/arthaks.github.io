## 你好，我是 Clay

> 一个从生物学"天坑"里爬出来的 Java 工程师，写了十年代码，依然热爱这个行业。
{: .prompt-info }

本科学的是生物学，毕业后为了生计转行写代码。因为 Java 入门友好、工作机会多，便选择了它作为起点。从小公司的通宵加班一路走来，后来有幸加入美团，在大厂里打磨了几年技术和视野。离开时没有选择"活水"，也没有跳去另一个大厂——厌倦了卷文档、卷汇报的氛围，选了一家节奏更平衡的公司。回头看，这个选择是对的。

如今，除了日常的后端开发工作，我把越来越多的精力投入到 **AI 工程化落地**这个方向，探索如何让 AI 真正融入研发团队的工作流。

## 我在这里写什么

- **AI 技术沉淀** — 系统记录我在 AI Agent、Prompt Engineering、Coding Agent 集成等方面的学习与实践
- **后端工程经验** — 十年 Java 后端积累的架构设计、性能调优、踩坑笔记
- **生活随笔** — 工作之外的思考，关于成长、家庭和生活

## 我还在写书

除了博客，我也在尝试将体系化的知识沉淀为**电子书**。目前已出版的第一本书是《Anthropic Engineering 学习指南》，聚焦 AI Agent 工程实践与企业架构设计。

所有书籍都陈列在 [**书架**](/books/) 页面，欢迎翻阅。

## 为什么写这个博客

工作十年了，想在这个世界上留下一些文字。

一方面是给自己——记录技术成长的轨迹，逼自己把"觉得懂了"的东西真正想清楚、写明白。

另一方面是给孩子——希望他将来能通过这些文字，了解他老爸是怎样思考、怎样工作、怎样生活的。
